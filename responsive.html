Responsive Design

Le responsive design c'est quoi? C'est vraiment utile ?





Le responsive design en quelques mots

C'est un procédé relativement récent qui consiste à adapter l'affichage du site en fonction du type d'appareil utilisé pour la consultation. La forte augmentation des consultations sur mobile a rendu nécessaire le développement de nouveaux outils pour assurer une bonne lisibilité et ergonomie, que le visiteur soit sur un ordinateur, un mobile ou une tablette.



Comment créer un site responsive?

Le principe est simple, il suffit principalement d'utiliser les media queries pour les css : en clair, cela permet d'interroger le navigateur pour savoir quelle est la taille en pixels de la zone d'affichage, le type d'écran, la position, puis de déterminer les paramètres de l'affichage en fonction de ces résultats : par exemple, le bloc B est à gauche du bloc A si l'affichage sur l'écran est supérieur à 1024 pixels, mais il passe en dessous dans le cas contraire. Mais le bloc C peut aussi disparaître parce que la fonction concernée n'est pas jugée utile sur les mobiles, le bloc D peut devenir D' avec de nouveaux contenus...etc. En pratique, les mises en page fluides requièrent une bonne connaissance du CSS 3 et le code d'un ancien site doit parfois être entièrement revu pour que l'affichage soit optimal pour tous les écrans.
